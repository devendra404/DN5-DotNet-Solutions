## 🎯 About This Repository

This repository contains hands-on exercise solutions completed during the **7-week DN 5.0 Deep Skilling program**, following the structured FSE (Full Stack Engineer) learning path for the **DotNet + Angular** track.

The program covers the complete engineering competency landscape across four major constructs — from core design concepts to production-grade deployment tools.

## 📚 Modules Covered

<details>
<summary><b>🧩 Module 1 — Design Patterns & Principles</b></summary>
<br/>

- **SOLID Principles** — SRP, OCP, LSP, ISP, DIP
- **Creational Patterns** — Singleton, Factory Method, Builder
- **Structural Patterns** — Adapter, Decorator, Proxy
- **Behavioral Patterns** — Observer, Strategy, Command
- **Architectural Patterns** — MVC, Dependency Injection

</details>

<details>
<summary><b>📊 Module 2 — Data Structures & Algorithms</b></summary>
<br/>

- Arrays, Linked Lists (Single, Double, Circular)
- Sorting — Bubble, Insertion, Heap, Quick, Merge
- Searching — Linear Search, Binary Search
- Time & Space Complexity, Big-O Notation

</details>

<details>
<summary><b>🗄️ Module 3 — Advanced SQL (SQL Server)</b></summary>
<br/>

- Window Functions — OVER(), PARTITION BY, RANK, DENSE_RANK
- CTEs, Recursive CTEs, MERGE, PIVOT, UNPIVOT
- Views, Indexes, Stored Procedures, Functions
- Triggers, Cursors, Exception Handling, Transactions

</details>

<details>
<summary><b>🧪 Module 4 — NUnit & Moq</b></summary>
<br/>

- Automated Testing fundamentals & Test Pyramid
- Unit Testing with NUnit — parameterized, setup, teardown
- Mocking with Moq — state-based & interaction testing
- Dependency Injection patterns for testability

</details>

<details>
<summary><b>🔗 Module 5 — Entity Framework Core 8.0</b></summary>
<br/>

- Code-First approach, DbContext, Migrations
- CRUD operations, LINQ queries, async patterns
- Relationships — One-to-One, One-to-Many, Many-to-Many
- Eager / Lazy / Explicit loading, AsNoTracking

</details>

<details>
<summary><b>🌐 Module 6 — ASP.NET Core 8.0 Web API</b></summary>
<br/>

- RESTful API design — Controllers, Routing, Action Verbs
- JWT Authentication & Authorization
- Middleware, Filters, Global Exception Handling
- Swagger/OpenAPI, Serilog logging

</details>

<details>
<summary><b>⚙️ Module 7 — Microservices Architecture</b></summary>
<br/>

- Microservices vs Monolith
- Inter-service communication — HTTP, gRPC, Messaging
- Service Discovery, JWT security, Health Checks
- Docker + Kubernetes deployment strategies, CI/CD pipelines

</details>

<details>
<summary><b>🅰️ Module 8 — Angular v20</b></summary>
<br/>

- Components, Data Binding, Lifecycle Hooks
- Directives, Pipes, Reactive & Template-driven Forms
- Routing, Lazy Loading, Route Guards
- Services, Dependency Injection, HttpClient
- RxJS Operators, NgRx State Management
- Unit Testing with Jasmine & Karma

</details>

<details>
<summary><b>🐛 Module 9 — Application Debugging</b></summary>
<br/>

- Chrome DevTools — DOM inspection, breakpoints, source maps
- VS Code Debugger — launch.json, watches, TypeScript debugging
- Debugging NgRx store and Angular services

</details>

<details>
<summary><b>🔀 Module 10 — Version Control (GIT)</b></summary>
<br/>

- Git fundamentals, branching strategies
- Remote repositories, pull requests, forking
- Gitflow, Feature Branch, Centralized workflows

</details>

<details>
<summary><b>🔁 Module 11 — DevOps & CI/CD</b></summary>
<br/>

- DevOps principles and practices
- Continuous Integration & Continuous Delivery
- Jenkins, GitHub Actions, GitLab CI/CD

</details>

<details>
<summary><b>🐳 Module 12 — Containerization (Docker)</b></summary>
<br/>

- Docker images, containers, volumes, networking
- Dockerfile, Docker Compose, Docker Engine
- Container orchestration with Kubernetes

</details>

<details>
<summary><b>🤖 Module 14 — Gen AI Fundamentals</b></summary>

- Generative AI concepts, history, and applications
- Prompt Engineering — Zero-shot, Few-shot, Chain-of-thought

## ⚡ Quick Start

**Prerequisites**

| Tool | Download |
|------|----------|
| Visual Studio Community | [Download](https://visualstudio.microsoft.com/vs/community/) |
| SQL Server Management Studio | [Download](https://learn.microsoft.com/en-us/ssms/download-sql-server-management-studio-ssms) |
| VS Code | [Download](https://code.visualstudio.com/download) |
| .NET SDK 8.0+ | [Download](https://dotnet.microsoft.com/en-us/download) |
| Node.js (for Angular) | [Download](https://nodejs.org/) |

```bash
# Clone
git clone https://github.com/devendra404/DN5-DotNet-Solutions.git
cd DN5-DotNet-Solutions

# Run a .NET exercise
cd "Week1/Design Patterns and Principles"
dotnet run
```
